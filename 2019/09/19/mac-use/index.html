总结一些mac使用技巧

快捷键

  • Shift + Command + 4 mac 电脑自带的截图快捷键，截图完毕之后会保存到桌面

删除账号重新设置

  • 开始时按住Command + S 或者是 Command + S + fn进入单一用户模式，就是一个命令行界面
  • 执行 /sbin/fsck -y 执行硬盘检测, 这一步可以省略
  • 执行 /sbin/mount -uaw加载文件系统
  • 执行 rm /var/db/.AppleSetupDone 删除初始化设置时的OSX生成的隐藏文件.AppleSetupDone
  • 执行 reboot 重启电脑，会再次进入重新设置页面

注意：有时候你会发现开机时，你明明按了Command + S 或者是 Command + S + fn可就是没有进入单一用户模式，
还是需要你输入密码，如果你知道密码，输入密码之后就能进入单一用户模式了，否则不能进入。就不要一直关机重启试了
